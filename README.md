# Aplicacion web para inventarios

### Descripcion

Te presento esta aplicacion web para el manejo eficiente de inventarios de productos en multiples tiendas (Ideal para tiendas de ventas).

### Tecnologias utilizadas

- HTML/CSS
- JavaScript
- PHP
- MySQL

### Principales funcionalidades

- Registros y logeo de usuarios en la aplicacion
- Registros de productos en la base de datos (Codigo, Nombre, Precio, Marca, Proveedor)
- Actualizar el inventario de un producto cada vez que se vende y se abastece la tienda de ese producto
- Consultar la cantidad en stock de un producto
- Mover inventario a otros almacenes
- Crear nuevos almacenes
- Consultar, editar y eliminar un producto
